SANGAM - Brand Identity for the Cultural Council, IIT Roorkee
=============================================================
Designer submission - Cult Open Projects 2026

CONCEPT
The Sangam Mark: six 45-degree color wedges (six creative streams) converge
around a shared nucleus and open toward the audience - reading at once as a
"C" for Culture and a stage aperture. Named for the confluence, echoing
Roorkee's Ganga Canal heritage.

FOLDER GUIDE
00_Brand_Guidelines.pdf ......... 11-page guidelines: strategy, mark anatomy,
                                  logo rules, color, type, applications
01_Logos ........................ Primary, secondary, monochrome (black/white),
                                  brandmark, profile icons - all SVG + PNG
02_Color ........................ Palette sheet with HEX + RGB values
03_Typography ................... Type system sheet (Poppins + Lato, rationale)
04_Social_Media ................. Announcement, event promo, general
                                  communication posts + Instagram grid preview
05_Mockups ...................... Hoodie (physical application), front + back
06_Motion ....................... 3.5 s logo reveal (MP4 + GIF)
07_Section_Extensions ........... Six section sub-brand lockups
08_Additional_Applications ...... Event pass + sticker pack

COLOR SPECIFICATIONS
Primary   Sangam Violet #6C35E0 (108,53,224) | Ink Black #16121E (22,18,30) | Paper White #FAF7F2 (250,247,242)
Secondary Riverline Blue #2E7CF6 (46,124,246) | Canal Teal #00B0A3 (0,176,163) | Raag Magenta #E8386D (232,56,109)
Accent    Genda Amber #FFB020 (255,176,32) | Diya Coral #FF6B4A (255,107,74)

TYPOGRAPHY
Primary: Poppins (Google Fonts) - logo, headlines, titles
Secondary: Lato (Google Fonts) - body, captions, long-form

All artwork is original and produced programmatically (vector-first).
